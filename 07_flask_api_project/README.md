# 🌐 07 - Flask API AI Deployment

It's time to build a web interface around AI code so users can connect! This module transitions from local scripting to hosting a Python Flask web server.

## 📖 Topics Covered
1. **`app.py`**: The central application loop and router setting up endpoints.
2. **`config.py`**: Configuration constants managing how the application is built.
3. **`requirements.txt`**: Run `pip install -r requirements.txt` to install `Flask` and `gunicorn`.
4. **`app/`**: Route implementations handling API traffic.
5. **`tests/`**: Component and unit tests for ensuring the server continues working in production!
