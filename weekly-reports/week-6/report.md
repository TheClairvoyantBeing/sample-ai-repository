## ✅ Tasks Executed

Implemented supervised machine learning algorithms, specifically Logistic Regression, utilizing the Scikit-learn library. Executed 70/30 train-test splits to prevent model overfitting and generated comprehensive classification reports detailing precision, recall, and F1-scores. Transitioned to backend API architecture by developing RESTful endpoints to serve data. Utilized the Python `requests` library for GET/POST methods and validated JSON payload responses using Postman. Finally, configured and tested local Large Language Models (LLMs) via Ollama and Hugging Face, managing local GPU requirements for AI model inference.
