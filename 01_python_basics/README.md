# 🐍 01 - Python Basics

Welcome to the start of your Python journey! This section covers the absolute foundational concepts of Python programming.

## 📖 Topics Covered

1. **`01_hello_world.py`**: Your first Python program, print statements.
2. **`02_output.py`**: Formatting output, f-strings, and print parameters (`sep`, `end`).
3. **`03_variables.py`**: Declaring variables, naming conventions, and the `type()` function.
4. **`04_datatypes.py`**: Core data types: strings (`str`), integers (`int`), floats (`float`), and booleans (`bool`).
5. **`05_operators.py`**: Arithmetic, comparison, and logical operators.
6. **`06_user_input.py`**: Getting input from the user and converting data types (`input()`, `int()`).
7. **`07_conditional.py`**: Control flow using `if`, `elif`, `else`, and ternary operators.
8. **`08_looping.py`**: Repeating blocks of code with `while` and `for` loops, plus `break` and `continue`.

## 🛠️ How to use
Run these files sequentially. Read the comments entirely as they contain key context!

```bash
python 01_hello_world.py
```
