# 🚀 06 - Projects, Envs, and Utilities

A collection of applied projects, configurations, and scripts to demonstrate end-to-end Python concepts.

## 📁 Contents

1. **`01_ascii_bot.py`**: An interactive terminal bot calculator putting basic Python concepts to work.
2. **`02_sentiment_analysis`**: A pre-trained HuggingFace transformers NLP pipeline application.
3. **`03_utils`**: Helpful functional Python utility files that can be imported across projects.
4. **`04_conda_envs`**: Example environment YAML descriptors for setting up Data Science and NLP environments using Conda.
