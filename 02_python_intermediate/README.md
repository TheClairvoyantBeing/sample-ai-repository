# 🐍 02 - Python Intermediate

Take the next step in Python programming! In this section, we cover abstractions, file handling, and organizing data inside scripts.

## 📖 Topics Covered

1. **`01_functions.py`**: Defining reusable blocks of code, parameters, `return`, `*args`, `**kwargs`.
2. **`02_structures.py`**: Advanced data structures: Lists, Tuples, Dictionaries, and Sets.
3. **`03_exceptions.py`**: Graceful error handling using `try`, `except`, `finally`, and `raise`.
4. **`04_modules.py`**: Importing built-in modules (`datetime`, `os`, `math`, `random`) to expand capabilities.
5. **`05_file_io.py`**: Reading, writing, and appending files locally (integrates with `../data/`).
6. **`06_csv_ops.py`**: Working with CSV (Comma-Separated Values) files using standard `csv.reader` and `DictReader`.

## 🛠️ How to use

```bash
python 01_functions.py
```
