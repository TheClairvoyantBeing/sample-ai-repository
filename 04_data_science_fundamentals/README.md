# 📊 04 - Data Science Fundamentals

This directory introduces the three titan libraries of Python Data Science: **NumPy**, **Pandas**, and **Matplotlib**. 

## 📁 Modules Included

* **`01_numpy`**: Efficient multi-dimensional array operations, reshaping, and statistical computing for ML payloads.
* **`02_pandas`**: DataFrame manipulation, creation, grouping, filtering, and data sanitization techniques.
* **`03_matplotlib`**: Training/Loss visualizations and plotting accuracy progressions.

### Setup Instructions

Before running the code in here, it is recommended to install the required libraries:
```bash
pip install -r 01_numpy/requirements.txt
pip install -r 02_pandas/requirements.txt
pip install -r 03_matplotlib/requirements.txt
```
