# 🔌 05 - Model Context Protocol (MCP)

This directory features progressive tutorials introducing the Model Context Protocol (MCP), a system allowing AI and LLMs to interact with external tools, APIs, and file systems natively!

## 📖 Topics Covered

1. **`01_what_is_mcp.py`**: Concepts, terminology, and architectures.
2. **`02_simple_mcp_server.py`**: A foundational MCP configuration without extraneous logic.
3. **`03_mcp_tools.py`**: Creating custom actions externalized to an MCP configuration.
4. **`04_mcp_resources.py`**: Bridging data sources and structured context objects to MCPs.
5. **`05_mcp_prompts.py`**: Prompt engineering specifically geared towards driving an MCP server efficiently.
6. **`06_mcp_client.py`**: Code allowing a Python app to speak directly to an MCP server instance.
7. **`07_real_world_example.py`**: A combined, applied demonstration handling real-world operations using the tools compiled across this module.
